## Introduction

This is a small and simple GUI tool to gather your computer's information using Python.

## Run

Upgrade pip then install required modules:

```shell
python -m pip install --upgrade pip
python -m pip install -r requirements.txt
```

then run:

```shell
python sysig.py
```
